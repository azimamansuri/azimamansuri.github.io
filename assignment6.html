Assignment 6: Final project proposal

In this assignment, we were tasked to create a project plan to outline our final project.



Concept

For my final project, I wanted to build off of the mediterranean tile concept I had for my molding project. My sister and I are creating a print shop and are in the process of creating ink prints that would be used as cards, posters or gifts. For the mold project, I created two turkish-inspired tile design molds to be used as stamps. For this final project, I’m hoping to create a cohesive collection to fit this theme.

I’m planning out the collection, but would love to incorporate some arabic calligraphy/writing into it as well to showcase the connectedness between the architecture, culture, and script of the mediterranean, and found some inspiration online to create a moodboard:


Through this project, I want to experiment with different shapes and patterns that I can create through different methods (rhino and grasshopper), and then print the mold which I plan to cast silicone into and then plaster as a final piece. These objects would be used as stamps for these prints.

Bill of materials

I will be using the remaining filament I have from class (since it will only be used for the mold of molds), oomoo, plaster, watercolor paper, black ink, and a roller to roll the ink onto the plaster stamp. I have rubber sheets if the plaster is too stiff to transfer the ink over to the paper to carve and use if needed. I don’t anticipate needing anything else that I don’t already have sourced!
